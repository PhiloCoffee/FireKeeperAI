# Fire Keeper AI Asset Library

Assets are grouped by intended UI role:

- `scene/`: environment and background images.
- `char/`: character-forward screenshots or portraits.
- `object/`: focused objects, props, and bonfire imagery.
- `brand/`: store/header/capsule art and title-style assets.
- `reference/`: contact sheets and review-only selection aids.
- `manifests/`: source and usage notes for downloaded or local screenshot assets.

Dark Souls assets in this folder are copyrighted game content from Steam store assets or local Steam screenshots. Use them for local fan prototypes only unless properly licensed.
